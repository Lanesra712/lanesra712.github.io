前言

既然需要在程序运行前将静态数据写入到缓存中，毫无疑问我们需要在程序运行前执行一些自定义功能的代码，那么在本章中，我将会介绍如何在 ASP.NET Core 项目中，实现在程序启动前执行某些特定功能的代码。
